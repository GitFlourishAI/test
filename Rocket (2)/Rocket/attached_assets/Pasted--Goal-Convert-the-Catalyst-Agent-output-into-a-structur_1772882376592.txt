## Data Model

```python
class CatalystReport(BaseModel):
    id: str  # UUID format (example: 1f0cf03d-04c9-4a65-827f-1ec644dbf6cf)

    title: str  
    # Example: "Fed Rate Decision: 50bp Cut Expected"

    description: str  
    # Detailed summary of the catalyst event

    sector_id: str  
    # Must match one of the following sector UUIDs

    source_type: str  
    # Allowed values:
    # policy | earnings | macro | commodity | sector_news

    source_name: str  
    # Example: "NVIDIA Earnings Call"

    source_url: str  
    # Example: https://bloomberg.com/markets/currencies

    detected_at: datetime  
    # Example: Mon Feb 23 2026 14:20:06 GMT+0800

    time_horizon: str  
    # Allowed values: 3M | 6M | 12M

    confidence_score: float  
    # Range: 0–100

    impact_magnitude: float  
    # Percent change estimate (example: 3.2)

    impact_direction: str  
    # positive | negative

    status: str  
    # active | archived

    shock_type: str  
    # Regulatory | Demand Shock | Supply Shock |
    # Policy Tailwind | Competition | Currency Shock | Monetary Policy

    evidence_summary: str  
    # Short explanation supporting the catalyst

    ticker: str  
    # Example: JNJ

    revenue_impact: float
    ebitda_impact: float
    valuation_delta: float
```
